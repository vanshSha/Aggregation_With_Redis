-- Aggregation_Stream_With_Redis

 * About 
    - In this Spring Boot project . I am doing aggregating some stream then sent into redis.
 
  * Tech Stack
   
   - Java Version 17
   - Maven Project
  
  * Docker Images 
   - Kafka
   - Zookeeper
   - KSQLDB
   - Redis

  * Dependencies
    -  Kafka
    -  Kafka Stream
    -  Redis
    -  Lombok 
 
  * How to run
   
    -  ./mvnw spring-boot:run. -> Spring Boot

    -  docker compose up -d -> Docker
